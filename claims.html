Condominium Building Claim Management Application

This Condominium Building Claim Management Application was built with Java Spring Boot and JPA for the backend, and ReactJS for the frontend.

It was the group assignment for Interactive Applications course.

On one hand, through an administrative user, it allows the registration, modification, and deletion of buildings, units, owners, and tenants. On the other hand, the application aims to enable its owner and tenant users to register claims regarding incidents in their own buildings or units.







They can add a description of the claim as well as photos. Once the claim has been created, they can review it. The administrator can change the status of a claim, indicating the actions taken.
