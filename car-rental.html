Car rental web application

This car rental web application was built with ASP.NET using Razor Pages

The system covers the small part of car rental like user sign up, sign and car search to booking.







The main page allows users to see available cars and manage rentals one. It also provides access to their reservations, booking history, and profile settings.

The admin user can manage inventory, reservations, and users.
